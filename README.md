## Application allows to compare two documents (JSON or YAML formats) and get the difference between them.
 

## Setup
```sh
make build
```

## Run
```sh
make run
```

## Run tests
```sh
make test
```

## Run checkstyle
```sh
make lint
```

## Check update dependencies and plugins
```sh
make update-deps
```
